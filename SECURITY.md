# Security Policy

To report a security vulnerability, please create a private vulnerability report directly through the GitHub Security tab. 

For sensitive disclosures that cannot be reported via GitHub, contact us on Discord (@plun1331, ID 545064119326212106 through https://discord.gg/skykings) 
with a clear description of the issue, reproduction steps, and impact. We aim to acknowledge all reports within 48 hours and work toward a remediation plan as quickly as possible.
